用法：if可以判断真假表达式，或者是能判断是否为空的数据结构，在表达式为真时运行的代码
书写方式：
在所有需要在if里面运行的代码，都需要添加至少一个空格作用缩进，一般约定缩进用4个空格，即一个tab，目的是方便阅读，同时满足python的规则，一旦退出缩进，程序或新的代码就不属于这个if
