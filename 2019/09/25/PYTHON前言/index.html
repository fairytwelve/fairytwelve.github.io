内置函数

input（）；用于接收用户输入的内容

3.1 整数、浮点、变量

整数和浮点数

python中的整数和数学中定义的整数是一样的。
python中的浮点数和数学中定义的小数是一样的。
在python中，我们可以使用print函数来答应一个整数或者浮点数。
整数的加减乘除运算可以在print中直接运行，同时还支持通过括号的方式来改变运算的优先级。
注意：在python中做除法运算时，返回浮点数。
在浮点运算中，如0.3+0.1不是0.4，而是一段很长的浮点数。这不是Python的问题。这是由浮点数的存储机制导致的。
注意：不应该直接用python做精确运算，但是进行日常精度不高的四则运算时没有问题的。

变量

变量是可以变得。可以理解成一个存放其他数据的盒子，使用变量可以减少重复的输入。
len=10
width=2
hei=5
d=len * width
s=d*hei
print(d)
print(s)

一个合法的变量在等号左边为变量，右边为常量值。其中等号就是赋值的意思。
补充：变量的类型完全由赋值的结果来确定，结果是整型就是整型；结果是字符串就是字符串

3.2字符串、列表、元组

1.字符串

在python中字符串是一种比较常见的数据类型，任何被单引号或双引号括起来的内容，都可以被认为是字符串，字符串也可以赋值给变量
字符串的内容可以是中文，英文、数字、空格以及其他字符组成。
字符串形式的数字和普通数字是不一样的。
int=123
string=‘123’
这两个数是一样的。

2.列表（List）

数据类型的表现形式为list=[]
列表在python中是一种容器类型，由方括号括起来的数据组成，里面的数据可以是整数、浮点数、字符串，也可以是另一个列表或其他的数据结构。
列表中的每一项叫做列表的元素，每一个元素之间用英文逗号隔开。

3.元组（tuple）

数据类型的表现形式为：tuple=（）
元组也是python中的一种容器类型，由小括号括起来的数据构成，外型和列表非常像，只不过列表为方括号，元组为小括号。
“元”和“二元一次方程”中的元是一个意思，“组”就是组合的意思。
如：tuple=（1,2,3,4,5）#元组中有6个元素，都是数字
tuple=（“123”，”知了堂“）
tuple=（）#空元组
tuple4=（“123”，[123,1,2],)
以上我们可以看到元组的支持类型》列表》字符串》整数

元组和列表的区别：
列表生成后，还可以在里面继续添加数据，也可以从里面删除数据
元组生成以后，不可以修改其中的整数、浮点数、字符，另一个元组，里面是列表的可以做修改。

3.3数据的读取

字符串、元组、列表三种数据结构的读取方式完全一样。

1-指定下标

在大多数的编程语言里面，下标都是从0开始的，python也不例外，也是从第0个元素开始的，第0个指的是元素的最左边的那个。
string
List=[“知”，“了”，“堂”]，第0个元素是“知”，第1个元素是“了”
tuple
想任取以上三种类型的元素，可以直接使用下面的表达式获取：
变量名[下标] 来获取对应下标的内容
下标：
范围：-长度+1 -0- 字符长度-1 如果有8个字符，那么字符范围是-7到7
下标中，如果是正值，则是正向取值，如
name=“我是知了堂”
name[1]=”是”
name[3]=”了”
如果是负值，则从最后一个元素开始取值。

2-切片操作

字符串通过切片以后结果还是字符串，列表切片以后还是列表，元组切片以后还是元组。
为什么切片？希望获得某个范围的内容时，需要用到切片操作。
切片格式： 变量名[开始位置的下标：结束位置的下标：步长]
其中开始位置的下标，结束位置的下标，步长都可以部分省略。
这三个参数对应的都是数字，范围不同。
开始位置的下标从0开始
结束位置的下标从0-字符长度的-1位，取值不包括下标结束的内容。
步长范围：1-字符长度的-1位，或-1（-1为倒序） 步长正值表示从左到右，负值表示从右到左

3.拼接和修改

字符串的拼接
string1=“你好”
string2=“，世界”
string3=string1+string2

元组与元组之间也可以相加，相加表示两个元组拼接在一起
tp1=（“123“，456）
tp2=（123，“456”）
tp3=tp1+tp2

列表与列表之间也可以相加，相加表示两个列表拼接在一起
以上的拼接原理三个数据类型都市一样的，掌握一个，其他通用。

修改内容
字符串重新赋值即可修改。
元组和列表的修改采用下标完成 变量名[下标]=新的值

元组是不可变容器，列表为可变容器，所以append（）只能作用于可变容器。
