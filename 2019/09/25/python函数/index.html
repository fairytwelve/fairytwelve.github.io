4·python 函数

4.1 什么是函数？

函数，就是一套定义好的流程，可以包含输入和输出，比如：输入数据，返回结果等等，在现实生活中，我们都被一个一个的函数所包围，可以体现在方方面面，如厨师的菜谱，如果厨师A把炒菜的过程写成一个教程，里面包含所有必须的输入条件，比如油温、盐等等，然后有过程中需要的时间，步骤详细罗列，我相信，这样的教程，只要换一个人再照着做一次，不说100分，80分也是可以的。所以厨师写菜谱和程序员写函数是一样的，都是为了能够将重复过程模块化，并可以随意调用执行，且结果还相当。

两个函数之间是可能相互独立的，也可能一个函数的输入是另一个函数的输出，也可能是一个函数调用另一个函数。

4.2 函数的作用

例：现在我想得到两个房间在不同情况下温度统计信息，得到这两个房间的温度的和、差、积、商等内容。
1.不用函数
2.用函数
函数的作用：就是把这个过程，具有相同类别的内容、规格统一放在一个代码块中，并根据函数的执行过程，或业务的执行流程来调用不同的函数或增减函数。

4.3 函数的定义

在python中，定义一个函数我们采用顶格的def关键字来定义一个函数，一个函数结构一般包含以下内容：
def 函数名（参数1，参数2，参数n）：
函数体过程代码1
······
函数体过程代码n
return 返回值
以上为最大函数体结构组成
以下为最小函数体结构组成
def 函数名（）：
函数体过程代码1
······
函数体过程代码n

返回值：一个函数可以返回至少1个返回值，可以人为的指定任务类型的返回数据，如果没有人为指定，那么返回值为none，返回值得个数可以是1个也可以是多个，函数的返回除了是常见的数据类型外，还可以是另个函数。
一个函数可以没有return语句，也可以有一个return语句，还可以有多个return。

在以下几个方面，return是等价的
1.没有return
2.return（只有return，后面不跟任务的变量和结果）
3.return none
在函数中，可以使用return将里面的结果返回出来，但代码一旦运行到return，那么函数就会结束，return后面的代码就不会被执行

函数的定义注意事项和书写的注意事项：
在一个python工程中，应该保证每个函数的名字是唯一，函数体就是这个函数执行的一系列的操作。操作可能只有一行代码，也可能是多行。
一个函数原则上只能做一件事情。同时python的编码规范定义，一个函数的函数体最多不能超过20行代码，如果一旦超过20行，说明这个函数做了不止一件事情。如果超过，那么应该将这个函数拆解成多个小函数。这里也暗示函数体内是可以调用其他函数的。

4.4 函数的调用

例：接收由用户输入的通过逗号分隔的两个非零整数参数，计算这两个数字的和、差、积、商，并返回给用户。

4.5 函数的参数

例：请输入由逗号分隔的两个非零整数
def getInput():
input_str=input(“请输入由逗号分隔的两个非零整数：”)
#分隔函数为python自带的内置函数
a_str,b_str=input_str.split(‘,’)
return int(a_str),int(b_str)

以上是改良前的代码，函数是没有参数的

def getInput(split_chat):
input_str=input(“请输入由{}分隔的两个非零整数：”.format(split_chat)
#分隔函数为python自带的内置函数
a_str,b_str=input_str.split(split_chat)
return int(a_str),int(b_str)
