NAT学习

2019年08月21日 251 字 大概 1 分钟 网络，NAT

NAT：网络地址翻译。用于将私有地址转成公有地址进入外部Internet上网等操作。还有安全策略，将局域网隐藏在广域网中。

回顾：私有地址有三类：
A类：10.0.0.0-10.255.255.255 2^24-2个有效地址
B类：172.16.0.0-172.31.255.255
C类：192.168.0.0-192.168.255.255

概述

Network Address Translation，网络地址翻译

  • 相关术语：
    内部本地地址（Inside Local IP address）
    内部全局地址（Inside Global IP address）
    外部本地地址（Outside Local IP address）
    外部全局地址（Outside Global IP address）
  • 转换类型：
    静态NAT
    动态NAT
    过载（Overload）
  • 一个IP地址不是本地就是全局的；
    本地IP地址在内部网络是可见的；

    端口地址转换（PAT）

    PAT在内部全局地址上使用唯一的源端口号区分各个转换的连接
    动态PAT可以实现地址复用
    静态PAT可以实现端口重定向
